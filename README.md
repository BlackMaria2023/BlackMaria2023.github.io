# BlackMaria2023.github.io

Se trabajó en un navbar con un logo y favicon. Se listaron diferentes secciones con smooth scrolling dirigiendo a las respectivas partes de la página.
Luego se agregó un carrusel de 3 imágenes principales para introducir la página Viajes Chile.
En tercer lugar se agregó una sección de presentación con tres párrafos que desaparece en dispositivos pequeños.
En cuarto lugar se incluyó una sección con cuatro tarjetas descriptivas y sus imágenes.
Después, un formulario de contacto, donde al ingresar el correo se muestra un mensaje con tooltip.
Finalmente se concluye con un footer con íconos y sus links respectivos.
